❤️ Heart Disease Prediction

📌 About the Project

This project is a Machine Learning web application that predicts whether a person is likely to have heart disease based on medical input values.

The application allows users to enter health details and instantly receive:

Prediction result (Disease Present / Not Present)

Prediction probability

The web interface is built using Streamlit and deployed online.

🧠 How It Works

The user enters medical details in the application.

The input data is scaled using a trained scaler.

A trained Machine Learning model makes the prediction.

The result and probability are displayed on the screen.

📊 Input Features Used

Age

Sex

Chest Pain Type

Resting Blood Pressure

Cholesterol

Fasting Blood Sugar

Rest ECG

Maximum Heart Rate

Exercise Induced Angina

ST Depression

Slope of ST

Number of Major Vessels

Thalassemia

🛠 Technologies Used

Python

NumPy

Pandas

Scikit-learn

Streamlit
